# Fashion MNIST Image Classification

## Project Overview

This project builds a **neural network** to classify grayscale images of clothing items into 10 categories, including shirts, shoes, bags, and more.

The model is trained on the **Fashion MNIST dataset**, which contains 70,000 images (28x28 pixels) of various clothing items.

## Dataset

* Fashion MNIST dataset is available via [Kaggle](https://www.kaggle.com/zalando-research/fashionmnist) or through TensorFlow/Keras datasets.
* Dataset is **not included** due to size limits.

## Objectives

* Explore image classification using neural networks
* Preprocess image data
* Build, train, and evaluate a neural network
* Visualize training performance and predictions

## Model Architecture

* Input layer: 28x28 grayscale images
* Hidden layers: Fully connected dense layers with ReLU activation
* Output layer: 10 neurons with softmax activation
* Loss function: Categorical Crossentropy
* Optimizer: Adam

## Example Output

Predicted labels for new images of clothing items.
Accuracy: ~90% (depending on hyperparameters)
